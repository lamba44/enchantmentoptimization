# Enchantment Optimization Tool

A simple web tool to help you find the most optimal enchantment combinations for your gear, without wasting levels, time, or trial-and-error.

**Check it Out:**  
https://enchantmentoptimization.vercel.app/

![Enchantment Optimization Tool](/src/assets/images/HomeScreen.webp)

## What this does

This tool calculates the most efficient order and combination of enchantments so you spend the least possible experience levels while applying them. Instead of manually testing combinations in-game or relying on guesswork, you get a clear and optimized result instantly.

## Key features

- Finds the lowest XP cost enchantment order
- Handles multiple enchantments and combinations correctly
- Fast, simple, and browser-based
- No installation or login required
- Designed to mirror real in-game mechanics as closely as possible

## Why use this

Enchanting can quickly become expensive and confusing, especially with multiple books involved. This tool saves you resources by:

- Reducing wasted XP levels
- Preventing costly mistakes
- Making enchantment planning straightforward

Just open the site, input your enchantments, and get the best possible result.

## How to use

1. Visit the live link above
2. Enter the enchantments you want to apply
3. View the optimal order and total cost
4. Apply it in-game with confidence
